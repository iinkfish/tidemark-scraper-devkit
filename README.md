## What this is
This is a little test environment to test the scraper files for the tidemark orchestrator. 
The [tidemark](https://github.com/iinkfish/tidemark) orchestrator is a scraper scheduler which writes it's results to a postgres DB. 

## How to use
1. Clone the repository
2. Install node dependencies
```
npm install
```
4. Run the scrapers (all included scrapers should work and produce an output)
```npm start```
